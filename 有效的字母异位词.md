思路很简单，统计每个元素个数，判断是否相等就行
```
class Solution:
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        dic_s = {}
        dic_t = {}
        for key in s:
            dic_s[key] = dic_s.get(key, 0) + 1
        for key in t:
            dic_t[key] = dic_t.get(key, 0) + 1

        return dic_t == dic_s
```
