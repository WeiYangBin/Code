
#### 要求使用空间复杂度为 O(1) 的原地算法。

```
class Solution:
    def rotate(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: void Do not return anything, modify nums in-place instead.
        """
        for i in range(k):
            nums.insert(0,nums[len(nums)-1])
            del nums[len(nums) -1 ]
            
        print(nums)
```
