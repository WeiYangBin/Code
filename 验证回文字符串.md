算法思路就是判断字符串与数字isalnum，然后lower()方法转化为小写，最后自需利用列表的特征反转[::-1]判断即可
```
class Solution(object):
    def isPalindrome(self, s):
        """
        :type s: str
        :rtype: bool
        """
        forward_lis = []
        for i in range(len(s)):
            if s[i].isalnum():
                forward_lis.append(s[i].lower())
                
        return forward_lis == forward_lis[::-1]
```
