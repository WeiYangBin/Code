算法思路就是将字符串中数字和字符取出，然后通过将字符串反向与正向判断相同为Ture反之false
```
class Solution(object):
    def isPalindrome(self, s):
        """
        :type s: str
        :rtype: bool
        """
        forward_lis = []
        for i in range(len(s)):
            if s[i].isalnum():
                forward_lis.append(s[i].lower())
                
        return forward_lis == forward_lis[::-1]
```
