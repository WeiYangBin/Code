算法思路就是将字符串中数字和字符取出，然后通过将字符串反向与正向判断相同为Ture反之false
```
class Solution:
    def isPalindrome(self, s):
        """
        :type s: str
        :rtype: bool
        """
        a = []
        for i in range(len(s)):
            if s[i].isalnum():
                a.append(s[i].lower())
        b = a[::-1]
        if b == a:
            return 1 > 0
        else:
            return 1 < 0
```
