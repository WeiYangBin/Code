解题思路：先将数组内序列还原成整数，然后在对还原出的整数加一，最后将加一后的整数转为str,利用列表的切片方法提取出每一个字符，然后强转int就好了

```
class Solution:
    def plusOne(self, digits):
        """
        :type digits: List[int]
        :rtype: List[int]
        """
        x = len(digits)-1
        s = 0
        d = []
        for i in range(len(digits)):
            s = s + digits[i]*(10**x)
            x = x - 1
        s = s + 1
        s = str(s)
        d = []
        for i in range(len(s)):
             d.append(int(s[i:i+1]))
        return d
```
