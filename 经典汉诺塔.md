#### 经典汉诺塔算法

```
def hanoi(n, a, b, c):
    """
    我们对柱子编号为a, b, c，
    如果a只有一个圆盘，可以直接移动到c；
    如果a有N个圆盘，可以看成a有1个圆盘（底盘） + (N-1)个圆盘，
    首先,需要把 (N-1) 个圆盘移动到 b，
    然后，将 a的最后一个圆盘移动到c，再将b的(N-1)个圆盘移动到c。
    """
    if n == 1:
        print(a, '-->', c)
    else:
        hanoi(n - 1, a, c, b)
        print(a, '-->', c)
        hanoi(n - 1, b, a, c)
hanoi(5, 'A', 'B', 'C')
```
