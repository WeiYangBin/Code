解法一： 简单思路，求个数 但是这个方法时间复杂度较慢
```
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        if len(nums) == 1:
            return nums[0]
        dic = {}
        for i in nums:
            if i in dic:
                dic[i] += 1
                if dic[i] > len(nums) / 2:
                    return i
            else:
                dic[i] = 1
            
```
解法二：直接利用sort排序取中间值
```
class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        return sorted(nums)[len(nums)/2]
            
```
