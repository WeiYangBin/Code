解法一： 简单思路，求个数 但是这个方法时间复杂度较慢
```
class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # return sorted(nums)[len(nums)/2]
        middle = len(nums) / 2
        d = {}
        for i in range(len(nums)):
            if nums[i] in d.keys():
                d[nums[i]] += 1
            else:
                d[nums[i]] = 1
        
        for key in d.keys():
            if d[key] > middle:
                return key
            
```
解法二：直接利用sort排序取中间值
```
class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        return sorted(nums)[len(nums)/2]
            
```
