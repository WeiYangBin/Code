```
class Solution:
    def firstUniqChar(self, s):
        """
        :type s: str
        :rtype: int
        """
        dic = {}
        for key in s:
            dic[key] = dic.get(key, 0) + 1
        for i in range(len(s)):
            if dic[s[i]] == 1:
                return i
        return -1
```
